About Us

Kami adalah tim G25 Online Academy Digital Talent Scholarship 2020 yang bersemangat dalam front-end web development. Kami percaya pengembangan front-end web bukan hanya tentang penulisan kode atau penataan halaman web, melainkan tentang menciptakan pengalaman yang memikat dan bermakna bagi pengguna.

HTML CSS Development

Tim G-25 mempunyai front-end web developer yang berbakat dalam membangun responsive website menggunakan HTML5 dan CSS3.

UI-UX Designing

Kami membangun desain yang dikodekan dengan baik agar tampilan responsive pada cross-platform serta mudah digunakan (user friendly).

Javascript Development

Tim developer kami berupaya memberikan solusi inovatif JavaScript seperti jQuery, NodeJS, AngularJS, dan ReactJS.

Dedicated Developers

Tim G-25 mempunyai developer yang selalu berupaya memberikan tampilan produk profesional untuk pengalaman pengguna yang lebih baik.
